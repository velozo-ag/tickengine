Description

This API will help you manage all aspects of a mid-sized cinema, including sales, ticketing, employee management, and screening scheduling.

Entities

For more information on utilities, please navigate to the following links:

  • [cinema]: Contains information about each cinema center. Every cinema contains the rest relavant information.
  • [user]: Users in the system (sales, managers, it_adm), associated with a specific cinema.
  • [screen]: Each cinema contains many screens (movie theaters).
    • [seat]: Every movie theater contains individual seats.
  • [screening]: Films that will be shown in the cinema.
    • [show]: Individual screening programming associated to a screen.
    • [movie]: All the information about the movie.
  • [sale]: Each sale will include tickets associated with a screening or film showing:
    • [ticket]: Contains all the necessary information about the show, including seat, schedule, screen.
